# 🏡 Home Decoration Store

A Django-based ecommerce web application for browsing and purchasing beautiful home decoration products online.

## 🌟 Project Overview

Home Decoration Store is a personal project built to demonstrate full-stack web development skills using Python Django. The platform allows users to explore a wide range of home decoration products, add them to cart, and proceed through a smooth checkout experience.

## 💡 Key Features

🛍️ Browse a wide range of home decoration products  
🛒 Add to cart functionality  
💳 Checkout process  
🖼️ Product images and media display  
📱 Clean and simple user interface  

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| 🐍 Python Django | Backend web framework |
| 🎨 HTML, CSS, JavaScript | Frontend design |
| 🗄️ SQLite | Database |
| ☁️ PythonAnywhere | Deployment platform |

## 🚀 How It Works

1. User visits the store and browses products
2. User adds favourite products to cart
3. User proceeds to checkout
4. Order is placed successfully

## 🚀 Getting Started

### Installation
1. Clone the repo
```bash
   git clone https://github.com/FouziaRiaz/home-decoration-store.git
```
2. Create virtual environment
```bash
   python -m venv venv
   venv\Scripts\activate
```
3. Install dependencies
```bash
   pip install django
```
4. Run migrations
```bash
   python manage.py migrate
```
5. Start the server
```bash
   python manage.py runserver
```
6. Open browser and go to `http://127.0.0.1:8000`
